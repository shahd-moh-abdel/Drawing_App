# Drawing App
A simple drawing application built with [raylib](https://www.raylib.com) in C.

## Features 
- Draw with your mouse
- select colors
- change brush size
- save your art work
- clear the canvas

## Build & Run

### Prerequisites
- [raylib](https://www.raylib.cpm) instlled
- GCC 

### Build 
```sh
bash build.bash
```

### Run 
```sh
./drawing_app
```
